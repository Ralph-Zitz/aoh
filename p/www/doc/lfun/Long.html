LFUN

string Long ()
string Long (string what)

DESCRIPTION

Returns the long description of this object.

Argument
what: for 'look at' commands, this is the id of the object looked at.

Result
A string with the complete long description of this object, but without content.

The result is catenated from P_LONG and the extra data associated with P_LONG. Actual objects may redefine Long() as they please to.

EXAMPLE

  write(obj->Long());

IMPLEMENTED

/std/thing/description
