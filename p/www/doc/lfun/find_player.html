/doc/efun/find_player

SYNOPSIS

object find_player(string str)

DESCRIPTION

Find a user with the name str. Users are found even if they are invisible or link dead. Monsters are not found.

This function uses the name that was set by set_living_name(). This is usually done automatically in player.c.

EXAMPLE

	object ob;
	ob = find_player("deepthought");

HISTORY

Since 3.2.1@40, this is a simulated efun. The handling of living names is no longer part of the driver.
