/doc/efun/query_ip_number

SYNOPSIS

string query_ip_number(object ob)
string query_ip_number(mixed & ob)

DESCRIPTION

Give the ip-number for the current user or the optional argument ob.

If ob is given as reference, it will upon return be set to the struct sockaddr_in of the queried object, represented by an array of integers, one integer per address byte
ob[0.. 1]: sin_family
ob[2.. 3]: sin_port
ob[4.. 7]: sin_addr
ob[8..15]: 0

HISTORY

The return of the struct sockaddr_in was introduced in 3.2.1@81.
