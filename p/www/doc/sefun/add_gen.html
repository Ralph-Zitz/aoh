/doc/sefun/add_gen

SYNOPSIS

string add_gen (string text)

DESCRIPTION

The approbiate genitiv 's' is added to <text>. Result is the possibly modified string.

EXAMPLE

    add_gen("one car")   --> "one car's"
    add_gen("two cars")  --> "two cars'"
    add_gen("Softbyte")  --> "Softbyte's"
    add_gen("Qtx")       --> "Qtx'"
    
  BUT
       
    add_gen("Qtx house") --> "Qtx house's" !
