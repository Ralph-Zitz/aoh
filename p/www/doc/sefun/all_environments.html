/doc/sefun/all_environments

SYNOPSIS

object e* all_nvironments (void|object obj)

DESCRIPTION

Return an array with all nested surrounding objects of <obj>, starting with the first (the one also returned by environment()). If no <obj> is given, the current object is assumed.

If no enviroments exist, an empty array is returned.
