/doc/sefun/fold_objlist

SYNOPSIS

string * fold_objlist (object * what)
string (*) fold_objlist (object * what, int impl)

DESCRIPTION

Sort the objects given in <what> by their InvShort() and return an array of their Short()s, with with matching descriptions collated as '<short>(<count>)'.

If <impl> is non-zero, the result is a string "<sh1>, ... ,<sh2> and <sh3>" of the strings from the array.
