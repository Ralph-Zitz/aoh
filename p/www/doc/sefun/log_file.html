/doc/sefun/log_file

SYNOPSIS

void log_file(string file, string message)

DESCRIPTION

Append a message to a log file. All log files are in the directory /log. '/log' is automatically prepended to the file name.

EXAMPLE

        log_file("foobar.err", "something went wrong\n");
