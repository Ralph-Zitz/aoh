SYNOPSIS

mixed *remove_alist(mixed key, mixed *alist)

DESCRIPTION

Remove the key and data specified by key of the alist and return the new alist.

EXAMPLE

        mixed *alist;
        ...
        alist=remove_alist("the magic key",alist);
