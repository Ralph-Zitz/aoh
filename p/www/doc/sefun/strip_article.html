SYNOPSIS

string strip_article (string text)

DESCRIPTION

Any "a ", "an " or "the " is removed from the start of <text> if necessary. The remainder is returned.

EXAMPLE

    strip_article("a cart")  --> "cart"
    strip_article("the gem") --> "geb"
