SYNOPSIS

string QueryShort()

LAST UPDATED

Pepel, Mon Sep 14 20:19:33 MET DST 1992

DESCRIPTION

All objects must have a short description. This function returns a short string describing the thing. The string must be usable in every sentence. QueryShort() will be called from descriptive texts like

Deepthought gives bottle of beer to beggar.

All three nouns, Deepthought, bottle of beer and beggar are just calls to QueryShort() in the specific object.

If the first word of the short description is not a noun or proper name, it must be in lower case (!).
