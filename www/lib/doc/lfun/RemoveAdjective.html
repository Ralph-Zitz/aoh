string * RemoveAdjective (string ad)
string * RemoveAdjective (string * ad)

DESCRIPTION

Remove one or more adjective strings from this object's P_ADS.

Argument
ad: a single string, or an array of strings to remove.

Result
P_ADS' new content.

EXAMPLE

  RemoveAdjective ("bent");
  RemoveAdjective (({"raving mad", "howling"}));
