LFUN

protected void emit_light (int l)

DESCRIPTION

Update this objects P_LIGHT level.

Argument l : the requested change of light.

The function updates P_LIGHT and informs the environment by a call to its light_from_inside(<l>).

Auxiliary function used to propagate light changes, like those caused by movements of lightsources. Don't call this manually.
