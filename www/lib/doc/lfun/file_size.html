/doc/efun/file_size

SYNOPSIS

int file_size(string file)

DESCRIPTION

Returns the size of the file in bytes.

Size -1 indicates that the file either does not exist, or that it is not readable for the calling object/user. Size -2 indicates that it is a directory.
