/doc/efun/remove_call_out

SYNOPSIS

int remove_call_out(string fun)
int remove_call_out(closure fun)

DESCRIPTION

Remove next pending call-out for function fun in this object. The time left is returned.

-1 is returned if there were no call-outs pending to this function.

EXAMPLE

To remove every pending call-out to MyTimer() 
	while (remove_call_out("MyTimer") != -1) /* continue */ ;

HISTORY

Removing a call_out to a closure was introduced in 3.2.1@45.
