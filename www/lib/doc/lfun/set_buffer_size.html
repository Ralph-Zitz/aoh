/doc/efun/set_buffer_size

SYNOPSIS

int set_buffer_size(int size)

DESCRIPTION

Changes the socket buffer size for this_interactive() to size, up to a preconfigured maximum, result is the old buffer size (or -1 on systems which aren't able to change the socket buffer). Modifying the buffer size may result in a better IO throughput, but can also worsen it.

HISTORY

Introduced in 3.2.1@34
