SYNOPSIS

int atoo (string str)

DESCRIPTION

Convert a <str>ing into the ordinal number it describes and returns it. If the conversion was not possible, a negative value is returned instead.

The function recognizes the ordinals "first" through "twentieth", the angloan representation ("1st", "2nd", etc) and the european ("1.", "3.").

EXAMPLE

    atoo("42,") --> 42
